Смысл задания:
Создать Rest API (JSON HTTP API)

что использовать в работе :
-Средства разработки: java 8
-Framework: Spring boot 1.3.0
- Бфза данных : MySQL
- протокол : HTTP, порт 80

Создать функционал (запросы):
1. Добавить нового пользователя.
-Передаем на сервер персональные данные пользователя (URL картинки, имя пользователя , емейл...)
-Сохраняем информацию в базе данных.
-Ответ сервера - уникальный АЙДИ нового пользователя.

2. Получение информации о пользователе.
- Передаем на сервер уникальный айди пользователя.
- читаем информацию из базы данных.
- ответ сервера - персональные данные пользователя.(выше)

3. Изминение статуса пользователя(Online, Offline).
- передаем на сервер уникальный айди пользователя и новый статус (Online, Offline).
- Изменяем статус пользователя.
- Ответ сервера - уникальный айди пользователя. новый и предыдущий статус.

Примечания: на сервере выполняется запрос к внешнему API/базе данных. 
Так как это упрощенное тестовое задание необходимо реализовать "заглушку" 
с имитацией обращения и задержкой по времени 5-10 сек.
